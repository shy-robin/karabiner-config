## 规则集

### SpaceFN

<https://ke-complex-modifications.pqrs.org/json/spacefn.json>

space + h -> left arrow
space + l -> right arrow
space + j -> down arrow
space + k -> up arrow

space + e -> escape
space + b -> space
space + u -> backspace
space + o -> enter
space + backspace -> forward delete
space + m -> `
space + , -> ~

space + [1234567890-=] -> F[1-12]
